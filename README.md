# Spring Boot + React + MySQL Full Stack Project

Spring Boot 기반 REST API와 React 프론트엔드를 사용하여 구현한
풀스택 웹 애플리케이션입니다.

백엔드는 Spring Boot로 REST API를 제공하고,
프론트엔드는 React를 통해 사용자 인터페이스를 구현했습니다.

이 프로젝트는 프론트엔드와 백엔드가 분리된 구조에서
REST API 기반으로 데이터를 처리하는 웹 서비스 구조를
구현하는 것을 목표로 개발했습니다.
